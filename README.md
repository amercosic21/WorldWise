# WorldWise // Keep track of your adventures

## About the Project

A React application that helps you keep track of your adventures around the world. Add cities you've visited, view them on an interactive map, and relive your travel memories.

## Features

- Add and manage locations you've visited.
- View your travel history on an interactive map.
- Keep notes for each location.

## Technologies Used

- React
- JavaScript
- CSS

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/amercosic21/WorldWise.git
   ```
2. Navigate to the project directory:
   ```bash
   cd worldwise
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
